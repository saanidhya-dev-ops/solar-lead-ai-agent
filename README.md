# 🔆 Solar Lead AI Agent

AI-powered system that qualifies and scores solar leads using AI, helping solar companies focus only on high-intent customers.

---

## ⚡ Problem

Solar companies lose time and money on:
- Low-quality leads
- Fake inquiries
- Unqualified customers
- Manual lead filtering

---

## 🧠 Solution

This AI agent:
- Analyzes incoming leads
- Extracts key data (budget, intent, location)
- Assigns a lead score (0–100)
- Classifies leads as HOT / WARM / COLD
- Helps sales teams prioritize follow-ups

---

## 🔄 Workflow

Lead Input → AI Analysis → Scoring → Categorization → Action Recommendation
